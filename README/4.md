## 路由Vue-Router
#### 1.路由基础介绍
1. 优缺点
  1. 优点: 
    - 用户体验好，不需要每次都从服务器重新获取，快速展现给用户
  2. 缺点: 
    - 不利于SEO；
    - 使用浏览器的前进，后退键的时候会重新发送请求，没有合理的利用缓存
    - 单页面无法记住之前滚动的位置，无法再前进、后退的时候记住滚动的位置
2. 路由
  1. vue-router用来构建spa
  2. `<router-link :to=""></router-link>`或者`this.$router.push({path: ""})`
  3. `<router-view></router-view>`
